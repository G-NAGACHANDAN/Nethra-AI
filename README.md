**Nethra AI** is an AI-driven quality assurance tool designed specifically for Micro, Small, and Medium Enterprises (MSMEs) in Tamil Nadu. By leveraging "Frugal Innovation," it transforms a standard smartphone into a powerful industrial inspection tool, eliminating the need for expensive high-end sensors.

## 🌟 The Problem & Our Solution

Small-scale industries (Textiles, Matchstick manufacturing, Automotive components) often lack the budget for automated quality control. **Nethra AI** bridges this Industry 4.0 gap by providing:
- **Low-Cost Entry:** No industrial cameras needed; works on mid-range smartphones.
- **Offline Reliability:** Works in rural workshops with zero internet connectivity.
- **Localized UI:** A bilingual interface (Tamil + English) designed for the local workforce.

---

## 🚀 Key Features

- **High-Efficiency Detection:** Utilizing **MobileNetV3**, optimized for mobile CPU performance and low latency.
- **Edge Computing:** Uses **TensorFlow Lite** for on-device inference, ensuring high speed and data privacy.
- **Bilingual Interface:** Fully localized UI in **Tamil and English** to ensure ease of use for factory operators.
- **Frugal Architecture:** Optimized for low power consumption and thermal efficiency on mobile devices.
- **TANCAM Aligned:** Developed with the vision of supporting Tamil Nadu’s manufacturing ecosystem.

---

## 🛠️ Technical Architecture

Nethra AI follows a "Mobile-First Edge" workflow to ensure minimal lag during inspection:

1.  **Image Acquisition:** Live stream via smartphone camera API.
2.  **Inference Engine:** Highly efficient **MobileNetV3** backbone tuned for classification and localization.
3.  **Optimization:** Model quantized and converted to `.tflite` format for maximum hardware acceleration.
4.  **Feedback Loop:** Real-time visual overlays + Haptic/Audio alerts for detected defects.
